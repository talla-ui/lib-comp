# T채lla UI component library

This package contains a library providing a selection of T채lla UI components. These can be used directly from the NPM package, or copy-pasted into your own application to reduce dependencies and to enable further customization.

## Playground

To see all components in action, you'll need Git, Node.js and NPM. Follow these steps:

- Clone the repository to your machine
- Run `npm install` to install all dependencies from NPM
- Run `npm run build` to compile the library to `lib/`
- Run `npm run dev` to host the playground app using Vite
- Use your browser to navigate to the URL that's displayed by Vite
